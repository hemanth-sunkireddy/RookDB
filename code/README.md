# Storage Manager

## Prerequisites
Install Rust:  
`curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh` (Linux/macOS)

## To Run Storage Manager
`cargo run`

Entry point: `src/main.rs`

## To Run Tests
`cargo test`

* Tests are written in `tests` folder.

### Generating Docs for code
`cargo doc --open`
